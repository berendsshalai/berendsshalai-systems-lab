# Social Content System

Social content is generated from real project releases, not empty motivational claims.

For each release produce:

- LinkedIn launch post, problem analysis, architecture explanation, lessons learned, privacy/audit post, demo post and educator post.
- X release announcement, technical thread, workflow breakdown, before/after process thread, repository update and learning resource thread.
- Instagram 6-10 frame carousel with alt text.
- Long-form article with problem, architecture, privacy, implementation, tests, result, limitations and reusable lessons.

No content is auto-published. Drafts require manual approval.
