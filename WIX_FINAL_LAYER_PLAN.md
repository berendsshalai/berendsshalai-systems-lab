# Wix Final Layer Plan

Wix is not the immediate work surface. It should present the evidence after the open-source GitHub ecosystem exists.

## When to Update Wix

Update the personal website only after a project has:

- Clean-room declaration.
- Passing tests.
- Passing privacy gate.
- Public GitHub repository.
- Versioned release or clear pre-release status.
- Working demo or documented local demo.
- Case study and course module.

## Wix Page Content Model

Each project summary should link to:

- GitHub repository.
- Release notes.
- Demo.
- Case study.
- Course module.
- Privacy model.
- Limitations.

## Current Blocker

Wix connector authentication requires reauthentication before live site edits can be made.
