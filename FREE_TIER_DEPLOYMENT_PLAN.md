# Free-Tier Deployment Plan

## Default

- GitHub repositories for source code.
- GitHub Pages for static demos, docs, course pages and case studies.
- GitHub Releases for versioned archives and sample files.

## Optional

Cloudflare Pages or Workers may be used only when static execution is insufficient and the implementation remains inside the free plan.

## Prohibited

- Paid hosting as a requirement for basic use.
- Uploading private documents to third-party services by default.
- Collecting learner or candidate data by default.
- Publishing to Wix before GitHub repositories, releases and case studies are stable.
