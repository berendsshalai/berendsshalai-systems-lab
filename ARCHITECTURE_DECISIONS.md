# Architecture Decisions

## ADR-0001: Clean Directory Per Public Project

Public repositories are created from new clean directories, never from private operational folders.

## ADR-0002: Configuration Before Claims

Labour, payroll, attendance and reconciliation rules are configuration values, not universal legal claims.

## ADR-0003: Synthetic Demonstrations by Default

Demonstrations use generated records with neutral names and identifiers.

## ADR-0004: Local-First Operation

Each project must offer local CLI and browser-only demonstration paths before optional hosted services.

## ADR-0005: MCP as Local Integration

MCP servers expose safe local analysis tools and synthetic resources. Live operational actions require explicit configuration and review.
